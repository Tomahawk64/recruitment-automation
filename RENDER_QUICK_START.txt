📦 STEP 1: DEPLOY BACKEND (Web Service)
═══════════════════════════════════════════════════════════════════════
1. Go to: https://dashboard.render.com
2. Click: "New +" → "Web Service"
3. Connect: GitHub repository "recruitment-automation"
4. Configure:
   ┌────────────────────────────────────────────────────────────────┐
   │ Name:              recruitment-automation-backend              │
   │ Region:            Frankfurt (EU Central)                      │
   │ Branch:            main                                        │
   │ Root Directory:    backend                                     │
   │ Runtime:           Node                                        │
   │ Build Command:     npm install                                 │
   │ Start Command:     npm start                                   │
   │ Instance Type:     Free                                        │
   └────────────────────────────────────────────────────────────────┘

5. Add Environment Variable:
   ┌────────────────────────────────────────────────────────────────┐
   │ Key: MONGO_URI                                                 │
   │ Value: mongodb+srv://RGdebtRelief:Admin123@rgdebtrelief...    │
   │        (Copy from backend/.env file)                           │
   └────────────────────────────────────────────────────────────────┘

6. Click "Create Web Service"
7. Wait for deployment... (~5 minutes)
8. 📝 COPY YOUR BACKEND URL: https://[your-service].onrender.com

═══════════════════════════════════════════════════════════════════════

🎨 STEP 2: DEPLOY FRONTEND (Static Site)
═══════════════════════════════════════════════════════════════════════
1. Click: "New +" → "Static Site"
2. Connect: Same repository "recruitment-automation"
3. Configure:
   ┌────────────────────────────────────────────────────────────────┐
   │ Name:              recruitment-automation-frontend             │
   │ Branch:            main                                        │
   │ Root Directory:    frontend                                    │
   │ Build Command:     npm install && npm run build                │
   │ Publish Directory: dist                                        │
   └────────────────────────────────────────────────────────────────┘

4. Add Environment Variable:
   ┌────────────────────────────────────────────────────────────────┐
   │ Key: VITE_API_URL                                              │
   │ Value: [Paste your backend URL from Step 1]                    │
   │        Example: https://recruitment-automation-backend.on...   │
   └────────────────────────────────────────────────────────────────┘

5. Click "Create Static Site"
6. Wait for build... (~3 minutes)
7. 🎉 YOUR APP IS LIVE!

═══════════════════════════════════════════════════════════════════════

⚙️ STEP 3: UPDATE BACKEND CORS (IMPORTANT!)
═══════════════════════════════════════════════════════════════════════
1. Go to Backend service settings → Environment
2. Add new variable:
   ┌────────────────────────────────────────────────────────────────┐
   │ Key: FRONTEND_URL                                              │
   │ Value: [Your frontend URL]                                     │
   │        Example: https://recruitment-automation-frontend.on...  │
   └────────────────────────────────────────────────────────────────┘
3. Click "Save Changes"
4. Backend will auto-redeploy
